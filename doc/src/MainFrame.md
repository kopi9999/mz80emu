# MainFrame

`MainFrame` is definition of main application window and is responsible for handling the window events.

## Buttons

| Button | Connected function |
| - | - |
| `File / Quit` | `OnQuit()` |
| `Stop clock` (pause icon) | `stopClock()` |
| `Run clock` (run icon) | `runClock()` |
| `Next tick` (arrow icon) | `nextTick()` |
